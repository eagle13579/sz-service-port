## ⚠️ 需要配置的项

### MySQL 数据库配置

**当前状态**: 未配置

**数据库连接信息**:
```
主机: localhost
端口: 3306
用户: root
密码: (空)
数据库: shuzhi_service_hub
```

**需要的操作**:

#### 选项 1: 使用 XAMPP (推荐初学者)

1. 安装 XAMPP: https://www.apachefriends.org/
2. 启动 XAMPP Control Panel
3. 点击 MySQL 的 Start 按钮
4. 访问 http://localhost/phpmyadmin
5. 创建数据库 `shuzhi_service_hub`
6. 导入 `unified-database-init.sql`

#### 选项 2: 使用独立 MySQL

```bash
# 1. 创建数据库
mysql -u root -p

CREATE DATABASE shuzhi_service_hub
  DEFAULT CHARACTER SET utf8mb4
  COLLATE utf8mb4_unicode_ci;

exit

# 2. 执行初始化脚本
mysql -u root -p shuzhi_service_hub < unified-database-init.sql

# 3. 导入测试数据 (可选)
mysql -u root -p shuzhi_service_hub < test-data.sql
```

#### 选项 3: 使用 Docker

```powershell
docker run -d -p 3306:3306 --name mysql ^
  -e MYSQL_ROOT_PASSWORD=root123 ^
  -e MYSQL_DATABASE=shuzhi_service_hub ^
  mysql:8.0

docker exec -i mysql mysql -uroot -proot123 ^
  shuzhi_service_hub < unified-database-init.sql
```
