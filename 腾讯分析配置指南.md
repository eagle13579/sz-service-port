## 🔧 配置步骤

### 第一步：注册腾讯分析

1. **访问腾讯分析官网**
   - 打开浏览器访问：https://mta.qq.com

2. **登录或注册**

3. **创建应用/网站**
   - 点击顶部菜单的「应用管理」或「移动应用」
   - 点击「+ 新建应用」
   - 选择「Web/网站」类型
   - 填写应用信息：
     - **应用名称**：数智服务港官网
     - **应用简介**：数智服务港官方网站

4. **获取统计代码**
   - 创建应用后，会显示应用列表
   - 找到你创建的应用，点击进入
   - 在应用详情页面，找到「统计代码」或「集成指南」
   - 记录以下信息：
     - **应用 ID (site_id)**：类似 `5001234567`
     - **渠道 ID (cid)**：类似 `5001234567`（通常与 site_id 相同）

## ✏️ 更新代码中的 ID

打开 `index.html` 文件，找到以下代码段：

```html
<!-- 腾讯分析 -->
<script>
var _mtac = {};
(function() {
    var mta = document.createElement("script");
    mta.src = "//pingjs.qq.com/h5/stats.js?v2.0.4";
    mta.setAttribute("name", "MTAH5");
    mta.setAttribute("sid", "YOUR_SITE_ID");
    mta.setAttribute("cid", "YOUR_CID");
    var s = document.getElementsByTagName("script")[0];
    s.parentNode.insertBefore(mta, s);
})();
</script>
```

将以下两个占位符替换为实际值：

1. **`YOUR_SITE_ID`** → 替换为你的 **应用 ID (site_id)**
2. **`YOUR_CID`** → 替换为你的 **渠道 ID (cid)**

**示例：**
```javascript
mta.setAttribute("sid", "5001234567");
mta.setAttribute("cid", "5001234567");
```

## 📈 查看统计数据

### 访问腾讯分析后台

1. 打开浏览器访问：https://mta.qq.com
2. 登录你的账号
3. 进入「应用管理」
4. 点击你的应用（数智服务港官网）
5. 即可查看各种统计数据

### 可查看的统计指标

腾讯分析提供以下数据：

- 📊 **访问统计**
  - PV（页面浏览量）
  - UV（独立访客数）
  - IP（独立 IP 数）

- 🎯 **用户分析**
  - 新老用户比例
  - 用户停留时长
  - 跳出率
  - 回访率

- 📱 **设备分析**
  - 访问设备类型（手机/平板/电脑）
  - 操作系统（iOS/Android/Windows/Mac）
  - 浏览器类型

- 🌍 **地域分析**
  - 访问者地区分布
  - 省份/城市统计

- 📅 **时间分析**
  - 每小时访问量
  - 每日/每周/每月趋势
  - 高峰时段分析

- 🔄 **页面分析**
  - 热门页面排行
  - 入口页面统计
  - 访问路径分析

## 🎯 高级配置（可选）

### 1. 自定义事件追踪

如果你想追踪特定按钮点击或其他事件，可以添加自定义事件：

```javascript
// 在按钮点击事件中添加
mta.click.stat('button_click', {
    'button_name': '立即体验',
    'page': 'home'
});
```

### 2. 设置页面浏览事件

```javascript
// 页面加载时记录
mta.page.stat({
    'page_title': document.title,
    'page_url': window.location.href
});
```

### 3. 追踪表单提交

```javascript
// 表单提交时
mta.click.stat('form_submit', {
    'form_name': 'contact_form',
    'page': 'about'
});
```

## 📝 注意事项

1. **延迟生效**
   - 数据统计有 5-10 分钟的延迟
   - 首次配置可能需要等待更长时间

2. **本地测试**
   - 在本地打开 `index.html` 时可能不会统计
   - 需要通过 GitHub Pages 部署的域名访问才会统计

3. **隐私保护**
   - 腾讯分析符合中国法律法规
   - 数据保存在国内
   - 符合个人信息保护要求

4. **性能影响**
   - 腾讯分析代码非常轻量
   - 对网站加载速度影响极小（< 10ms）

5. **跨域问题**
   - GitHub Pages 域名没有跨域问题
   - 统计代码可以正常工作

## ❓ 常见问题

### Q1: 配置后为什么看不到数据？

**A:** 可能的原因：
- 刚配置，需要等待 5-10 分钟
- 在本地预览，需要通过部署的域名访问
- ID 配置错误，请检查 site_id 和 cid
- 代码语法错误，检查浏览器控制台

### Q2: 如何排除自己的访问统计？

**A:** 腾讯分析暂不支持 IP 过滤功能。建议：
- 使用浏览器的隐私模式测试
- 或使用不同的设备访问

### Q3: 数据可以导出吗？

**A:** 可以。在腾讯分析后台：
- 找到「数据导出」功能
- 可以导出为 Excel 或 CSV 格式
- 支持自定义时间段导出

### Q4: 支持实时数据吗？

**A:** 支持。腾讯分析提供：
- 实时访客数
- 实时页面浏览
- 实时地域分布

### Q5: 可以添加多个网站吗？

**A:** 可以。一个腾讯分析账号可以管理多个应用/网站。

### Q6: 与 Google Analytics 有什么区别？

**A:**

| 对比项 | 腾讯分析 | Google Analytics |
|--------|----------|------------------|
| 数据位置 | 中国国内 | 海外 |
| 访问速度 | 更快 | 可能较慢 |
| 中文支持 | 原生中文 | 需要翻译 |
| 合规性 | 符合中国法规 | 需注意合规 |
| 免费 | 完全免费 | 免费版功能有限 |
| 实时性 | 支持 | 支持 |

对于面向中国用户的网站，推荐使用腾讯分析。
