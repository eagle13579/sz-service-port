## 🚀 快速开始

### 方案1: 使用模拟数据开发（推荐新手）

```bash
# 1. 启动前端
cd unified-frontend
npm install  # 首次运行
npm run dev

# 2. 访问
# 浏览器: http://localhost:5173

# 3. 开始开发
# 所有页面自动使用模拟数据
```

**优点**:
- ✅ 无需配置数据库
- ✅ 立即可用
- ✅ 适合UI开发

---

### 方案2: 使用真实API开发（推荐）

#### 步骤1: 启动MySQL

```cmd
# 方法1: 使用脚本（推荐）
启动MySQL容器.bat

# 方法2: 手动启动
docker run -d --name shuzhi-mysql ^
  -p 3306:3306 ^
  -e MYSQL_ROOT_PASSWORD=root123 ^
  -e MYSQL_DATABASE=shuzhi_service_hub ^
  mysql:8.0

# 等待25秒让MySQL启动
timeout /t 25
```

#### 步骤2: 初始化数据库

```cmd
# 初始化数据库结构
docker exec -i shuzhi-mysql mysql -uroot -proot123 shuzhi_service_hub < unified-database-init.sql

# 导入测试数据
docker exec -i shuzhi-mysql mysql -uroot -proot123 shuzhi_service_hub < test-init.sql
```

#### 步骤3: 启动后端

```bash
cd unified-backend
npm install  # 首次运行
npm run dev
```

后端将在 http://localhost:3002 启动

#### 步骤4: 启动前端

```bash
cd unified-frontend
npm run dev
```

前端将在 http://localhost:5173 启动

#### 步骤5: 配置前端使用真实API

编辑 `unified-frontend/src/utils/api.js`:

```javascript
const USE_MOCK_DATA = false  // 改为 false
```

重启前端服务

#### 步骤6: 测试

```bash
# 测试后端健康检查
curl http://localhost:3002/health

# 访问前端
浏览器: http://localhost:5173
```
