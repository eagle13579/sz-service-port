# 供应链平台开发项目

数智服务港供应链平台开发项目

## 项目结构

```
供应链平台开发/
├── frontend/          # 前端项目 (Vue 3)
├── backend/           # 后端项目 (Node.js + Express)
├── database/          # 数据库脚本
├── docs/             # 文档
└── README.md          # 项目说明
```

## 快速开始

### 后端启动

```bash
cd backend
npm install
npm run dev
```

### 前端启动

```bash
cd frontend
npm install
npm run dev
```
