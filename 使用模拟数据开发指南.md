# 数智服务港 - 使用模拟数据开发指南

## 📝 概述

由于MySQL数据库配置遇到问题，本项目提供了模拟数据方案，可以在不配置数据库的情况下开发和测试前端功能。

---

## 🚀 快速开始

### 启用模拟数据模式

编辑 `unified-frontend/src/utils/api.js` 文件：

```javascript
// 将这一行修改为 true
const USE_MOCK_DATA = true
```

### 重启前端服务

```bash
cd unified-frontend
npm run dev
```

现在前端将使用模拟数据，无需连接真实数据库！

## 💡 注意事项

1. **模拟数据仅用于开发**
   - 不要在生产环境使用
   - 生产环境需要真实数据库

2. **数据不会持久化**
   - 刷新页面后数据重置
   - 使用localStorage保存必要状态

3. **功能可能不完整**
   - 某些复杂功能可能需要真实后端
   - 根据需要补充模拟数据
