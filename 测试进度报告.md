**当前状态**: Docker命令执行遇到问题

**可选方案**:

#### 方案1: 使用Docker Desktop (推荐)
```powershell
# 1. 打开Docker Desktop
# 2. 运行以下命令
docker run -d --name shuzhi-mysql ^
  -p 3306:3306 ^
  -e MYSQL_ROOT_PASSWORD=root123 ^
  -e MYSQL_DATABASE=shuzhi_service_hub ^
  mysql:8.0

# 3. 等待15-20秒
# 4. 初始化数据库
docker exec -i shuzhi-mysql mysql -uroot -proot123 shuzhi_service_hub < unified-database-init.sql
```

#### 方案2: 使用XAMPP
1. 下载并安装 XAMPP: https://www.apachefriends.org/
2. 启动 MySQL 服务
3. 访问 http://localhost/phpmyadmin
4. 创建数据库 `shuzhi_service_hub`
5. 导入 `unified-database-init.sql`

#### 方案3: 使用本地MySQL
```bash
mysql -u root -p
CREATE DATABASE shuzhi_service_hub DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
exit
mysql -u root -p shuzhi_service_hub < unified-database-init.sql
```

- `unified-backend/.env` - 数据库密码已设置为 root123
