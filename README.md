# 数智服务港官网

产业带企业的一站式数智化出海助手

## 项目介绍

数智服务港致力于为产业带中小企业提供全套数字化及出海服务，涵盖出海培训、出海解决方案、增长运营、项目孵化及出海社群的一站式解决方案。

## 技术栈

- HTML5
- Tailwind CSS (通过 CDN 引入)
- Font Awesome (通过 CDN 引入)

## 本地开发

直接在浏览器中打开 `index.html` 文件即可预览网站。

## 部署

### GitHub Pages 部署

1. 将代码推送到 GitHub 仓库
2. 进入仓库 Settings -> Pages
3. 在 Source 下选择 "Deploy from a branch"
4. 选择 "main" 分支和 "/ (root)" 目录
5. 点击 Save，等待几分钟后即可通过 GitHub Pages 访问
