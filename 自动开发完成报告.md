## 🐛 故障排查

### PowerShell 脚本无法运行

**解决方法**:
```powershell
cd "c:/Users/56867/CodeBuddy/数智服务港/官网"
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\auto-setup.ps1
```

### Docker 未启动

**解决方法**:
1. 启动 Docker Desktop
2. 等待完全启动
3. 重新运行脚本

### 端口被占用

**检查端口**:
```powershell
netstat -ano | findstr "3306"
netstat -ano | findstr "3002"
netstat -ano | findstr "5173"
```

**解决方法**:
- 关闭占用端口的进程
- 或修改端口配置

### 服务无法启动

**解决方法**:
1. 检查 Docker 是否运行
2. 重新运行 `快速启动.bat`
3. 查看错误日志
